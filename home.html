Snap Package Available

December 11, 2023

While this isn't a new version, it is an alternate way of installing the application in Linux. For those with access to snap packages you can install AstraStack using

snap install astrastack

It is recommended that before installing the snap version that you first uninstall any existing version of the application by running the following from one of the existing Linux tar.xz downloads

sudo ./install.sh remove

You can view the application on snapcraft.io/astrastack.

New Version (2.4.0)

April 10, 2022

A new version has been released. While AstraStack is mainly intended for planetary stacking, it is still capable of some deep sky stacking which is why I have added some improvements for deep sky alignment and processing.

  • Added dering option (Sharpen)
  • Added dilate option (Align)
  • #CPU and auto-debayering override preferences are now saved
  • Upgraded to newer dependency versions
  • Small performance/memory usage improvements
You can download this release on the downloads page.
